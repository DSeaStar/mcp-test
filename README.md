# MCP Tool

这是一个用于处理Minecraft代码混淆和反混淆的工具。

## 功能特性

- 支持Minecraft代码的混淆和反混淆
- 命令行界面
- 支持批量处理文件
- 彩色输出支持

## 安装

1. 克隆此仓库
2. 安装依赖：
```bash
pip install -r requirements.txt
```

## 使用方法

```bash
python mcp_tool.py [命令] [选项]
```

### 可用命令

- `deobfuscate`: 反混淆代码
- `obfuscate`: 混淆代码
- `--help`: 显示帮助信息
